# Alfabetizacao Digital

Aplicacao web educacional criada como projeto de extensao em Analise e Desenvolvimento de Sistemas. O objetivo e oferecer atividades interativas para apoiar alfabetizacao digital, leitura e escrita em uma experiencia simples e acessivel.

## Objetivo

Construir uma interface de aprendizagem com exercicios progressivos, feedback imediato e recursos visuais para apoiar criancas, idosos e pessoas em processo de familiarizacao com tecnologia.

## Funcionalidades

- Atividades interativas para formacao de palavras e frases
- Feedback visual para acerto, erro e progresso
- Organizacao por telas e componentes reutilizaveis
- Interface responsiva
- Base preparada para evolucao de acessibilidade e novas atividades

## Stack

- React
- JavaScript
- Chakra UI
- Ant Design
- Framer Motion
- React Router

## Estrutura

```text
.
├── public/       # Arquivos estaticos

```

## Como rodar localmente

```bash
npm install
npm start
```

Build de producao:

```bash
npm run build
```

## Status

Projeto academico/educacional. Mantido como registro de evolucao em React e design de interfaces de aprendizagem.
